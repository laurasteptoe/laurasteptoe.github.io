CONTROLS: MOUSE(left click) and Space bar to wait one turn.

If a java error occurs:
--In Windows, go to control panel and search for java. In the java menu, go to security and turn the bar down to medium temporarily.
--In Mac, go to System Preferences and click on the java icon. In the java menu, go to security and turn the bar down to medium temporarily.
If your java is 32-Bit then also go to Advanced, and uncheck the 'enable the next generation java plugin' option.
(Or, just download the .jar or .exe files)
